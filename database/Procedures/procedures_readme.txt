allQueue()                  - returns all songs in the queue
topQueue(INT n)             - returns the 'n' top rows.  Takes 'n' as input parameter
close_table(INT table_num)  - closes a table with a specified number.  Takes the table number as the input parameter.  Outputs whether closing was successful and the number of requests made by table.
open_table(INT table_num    - opens a table specified by number.  Takes the table number as the input parameter.  Outputs whether opening was successful and the table's id number.
